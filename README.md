# Move Mox Manga / 移动Mox漫画 / Mox漫画整理ツール

[English](#english) | [中文](#中文) | [日本語](#日本語)

---

## English

### Overview
A simple batch script to organize manga epub files downloaded from Kmoe (kzo.moe / kmoe.one etc.) into series-named folders automatically.

### Features
- **Auto-detection** - Recognizes file naming patterns without manual configuration
- **Future-proof** - Works with any prefix when the site changes domain
- **Two naming styles supported**:
  - Bracket style: `[AnyPrefix][SeriesName]episode.epub`
  - Underscore style: `_AnyPrefix__SeriesName_episode.epub`

### Usage
1. Move `move_mox.bat` to the manga download folder
2. Double-click to run

## 中文

### 概述
一个简单的批处理脚本，自动将从 Kmoe（kzo.moe / kmoe.one 等）下载的漫画 epub 文件按作品名整理到对应文件夹中。

### 功能特性
- **自动识别** - 自动识别文件命名格式，无需手动配置
- **面向未来** - 网站更换域名或文件前缀后无需修改脚本
- **支持两种命名格式**:
  - 方括号型: `[任意前缀][漫画名]话数.epub`
  - 下划线型: `_任意前缀__漫画名_话数.epub`

### 用法
1. 将 `move_mox.bat` 移动到漫画下载文件夹
2. 双击运行即可

## 日本語

### 概要
Kmoe（kzo.moe / kmoe.one 等）からダウンロードした漫画 epub ファイルを、シリーズ名フォルダに自動整理するバッチスクリプトです。

### 機能
- **自動認識** - ファイル命名パターンを自動検出、手動設定不要
- **将来対応** - サイトのドメイン変更後もスクリプト修正不要
- **2種類の命名形式に対応**:
  - ブラケット型: `[任意プレフィックス][シリーズ名]話数.epub`
  - アンダースコア型: `_任意プレフィックス__シリーズ名_話数.epub`

### 使い方
1. `move_mox.bat` を漫画ダウンロードフォルダに移動
2. ダブルクリックで実行
